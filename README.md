#  Diwali Sales Analysis

##  About the Project

This project is based on the Diwali Sales dataset. The main objective of this project is to analyze customer purchasing behavior during the Diwali festival and find useful business insights using Python.

In this project, I performed data cleaning, data visualization, and exploratory data analysis (EDA) to understand which customers contribute the most to sales.

---

##  Tools & Libraries Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

##  What I Did

- Loaded the dataset
- Cleaned the data by removing unnecessary columns and null values
- Performed exploratory data analysis (EDA)
- Analyzed sales based on gender, age group, state, occupation, and product category
- Created different charts to visualize the results
- Found useful business insights from the data

---

##  Key Insights

- Female customers purchased more products than male customers.
- Customers between the age of **26–35 years** contributed the highest sales.
- Uttar Pradesh, Maharashtra, and Karnataka generated the highest revenue.
- Married women working in IT, Healthcare, and Aviation were among the top customers.

---

##  Conclusion

This analysis shows that businesses can improve their festive sales by targeting married women aged 26–35 years, especially in Uttar Pradesh, Maharashtra, and Karnataka.
